O que é o Box Model?

O Box Model é um conceito fundamental no CSS que define como os elementos HTML são exibidos em uma página web. Cada elemento é representado como uma caixa retangular, composta por quatro camadas principais:

  • Content (Conteúdo): O conteúdo real do elemento, como texto ou imagem.
  • Padding: Espaço ao redor do conteúdo, dentro das bordas do elemento. Cria uma área de respiro interna entre o conteúdo e a borda.
  • Border: Borda ao redor do padding e do conteúdo. Pode ser personalizada em espessura, estilo e cor.
  • Margin (Margem): Espaço ao redor da borda, separando o elemento dos outros elementos na página.

Ao ajustar o Box Model, podemos controlar o layout e o espaçamento dos elementos na página, tornando o design mais organizado e agradável.
